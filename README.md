## Installation

You can install the package via composer:

```bash
composer require astrogoat/locations
```

## Usage

```php
$locations = new Astrogoat\Locations();
echo $locations->echoPhrase('Hello, Astrogoat!');
```

## Testing

```bash
composer test
```
